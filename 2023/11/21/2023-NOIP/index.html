2023 NOIP 游记

2023/11/17

　　N老师说今天的任务就是放松，调整，于是就真的摆了一天。

　　这两天调了化学相对分子质量、网络连接这两道大模拟，还开了一半时间复杂度，没写完就被前来为我们鼓劲的杜老师和沃老师打断施法了。

　　剩下的时间都在玩纸牌、猫猫和 rich4。意外发现机房电脑可以运行我笔记本上没法运行的古老 rich4，完了可能三分钟又被F打断施法了。

　　中午还聚众开了一中午的盒，很开心。

　　晚上去试了机，学校包的大巴没走高架，遇上晚高峰堵了好久，到历二的时候已经挺晚的了，试了试键盘，手感还不错，尝试使用虚拟机但是没启动成，于是就放弃然后走了。

　　发现旁边那位 21 年就坐我旁边的名字特别牛逼的冷氏兄弟没有来，以为他自信不试机了来着。

　　出去以后看到 peterbei 老师了，然后想去找朱老师，但是朱老师已经走了，只好就这样告别众人跟妈妈去住酒店了。

　　酒店条件非常艰苦，点了鸡架和奶茶的外卖，但是胃口不太好也没吃多少，临睡觉还背了一下st表板子就睡了。

2023/11/18

早晨

　　早！心情还不错，但是看了我妈写的家长课堂小作业以后破防了。早餐吃了 mai 当劳，急匆匆地出发了。

　　到了门口发现学校车还没到，等了一会儿，面了朱老师，去拍合影了。

　　不出意外，HBaCr 喊口号还是喊的很大声，有心理准备了就没被吓到。

　　进场之前一直在听 HBaCr 絮絮叨叨，ljl 和 jjl 在前面笑，我和 lmh 在讨论 HBaCr 这些顺口溜对其他考生产生的影响。挺放松的，就开考了。

赛时

　　首先浏览了四道题的输入文件，听见隔壁老哥被字符串吓到倒吸凉气的声音，心里也有一种神秘的感觉。

　　开考。

　　看了题面，第一印象就是看不太懂，第四题挺好理解，前三道叙述的语言好像都有点繁琐抽象。平静了一下心情决定还是顺序开题。

　　A 又读了一遍以后十分钟就秒了。写了个优先队列排序，跑大样例有点卡，换了快读还能接受，就下一题了。

　　B 很神秘，越看越复杂，画了很多图，从最短路想到并查集想到连通性啥啥的，发现也弄不懂，放弃了。

　　C 也比想的要复杂，也考虑了很久，写了个假贪心，样例也过不了，琢磨得头大。这时还剩两个小时，就也放弃了，去做 D。

　　D 根据题面很快就写了个 DP，过程中发现按照自己的思路需要用到 vector 存 “在当前这一天的所有挑战”，但是我不会写 vector 遍历了，采用了一种下下策——二分优化遍历m的过程。调了快一个小时，但是过了预期 的大样例，感觉正解应该是线段树优化 DP，写起来也不值当，就开摆了，回头打暴力。

　　剩下一个小时，想去先写 B 的小范围暴力，但是发现性质要更好写一些，就想扣 的性质分， 写了没调出来，有点着急，仔细想了想感觉是想假了，遂怒打 暴力，一遍就过了。平静了一下，去把 C 前两个点的特判部分分写了。写完没测，心态有点崩，还剩下 15min，索性一边检查一边试图捡其他部分分，没查出啥问题，也没拿到别的分。

　　于是比赛就结束了。赛时的估分大概是 100+20+10+[36, 56)=[166,186)，还能接受，从出考场心情就淡淡的。

下午

　　很饿，坐了地铁还有很挤很挤的公交车，背了很重很重的包，去吃了很久很久都没去的黄氏许，然后累的瘫在床上瘫了一晚上，玩了一会儿猫猫、纸牌和扫雷，但是一局都没赢。

　　把码交了一份云斗，测出来是 166，不出所料的还能接受，但是看排名形势不是很能一等。

某个时刻

　　云斗统一测了山东代码，看的时候还没有测完，166还排130左右，高兴了一下就发现排名还在往后退。后来就退到了一个类似于之前S组排名的位置。

另一个时刻

　　小图灵出的分说我是179，确实很抽象，但是排名跟云斗的也差不多。

后来

　　小图灵改了排名，往前进了二三十名，但是分没变。

再后来

　　上洛谷测了一下，获得了令我震惊的183的高分。

2023/11/20

　　还是写了一下 vector 遍历，在洛谷跑到了 56pts。云斗没跑过去，依旧是原来的成绩。

　　我是小丑。原来 vector 可以用下标遍历。

　　无所谓了。

　　不知道该怎样结尾可以来收束这曲折但精彩的一年，备考前在机房学习的日子过得艰难但也充满乐趣，让我很难想象没有了oi以后的生活会是什么样子。

　　但是，NOIP 2023已经结束了，我们都该向前走了。

后记

　　最后，官方成绩是166，一等分数线是153。比预计的要低不少，这个成绩也算是给这一年纠结挣扎的付出画上的一个比较圆满的句号了。

　　竞赛课在这学期依旧要上，但是大抵在下学期就退役了，毕竟没有什么进队的打算。

　　这便是我的选择了。
